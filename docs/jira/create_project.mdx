# Create Sample Project in JIRA

1.	Click on **Create sample project**

    ![](images/jira16.png)

2.	Click on **Kanban software development**.  Click on **Next**

    ![](images/jira17.png)

3.	Fill in the **project name**.  Click on **Submit**

    ![](images/jira18.png)

4.	The sample project is created successfully

    ![](images/jira19.png)
