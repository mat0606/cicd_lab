Developer working on a change

  1. Find out the IP address for the Developer Workstation

  2. Putty into the IP address for the workstation. 

    username: nutanix
    password: retrieve the password from the trainer

  3. Run the following commands

    cd devops/web/src/css
    vi style.css

  4. Search for databox

  5. Change the background color. Saved the change.

  6. Run the following command 

    cd ~/devops

  7. Run the following command to commit the change into the repository

    git add .
    git commit -m ‘change css’
    git push origin master

  8. Observe the Jenkins pipeline. Drill into the devops pipeline.

  9. Wait for a few minutes for all the steps in the devops pipeline to complete.

  10. Login to Nexus OSS repository. Click on Browse. Click on CentralRepo

  11. The latest version 0.1.6 corresponds to the Jenkins Build No 6.
